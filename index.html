Game Server Data request

Link: /entry/?room=?type=?tag=?max=?hours=?minutes=?timeLowerBound=?keys_only=?json=
Method: GET
room: string
type: string, if type is no specified, match any type , if more than one type, match entry with type is one of those type (or condition)
tag: string, if tag is no specified, match any tag , if more than one tag, match entry with atleast those tag (and condition)
max: usigned int, range(0,1000), default=8 , 0 = get all possible entry
hours: usigned int, number of hours
minutes: usigned int, number of minutes
timeLowerBound: datime string, a lower bound for time filter
keys_only: bool, return only key if true
json: bool, return in json format
Description: Get number(max) of entries of a room that have specified tag from time distance from hours,minutes ago to now and have created time greater than timeLowerBound.

Link: /entry/?room=?content=?tag=?tag=...
Method: POST
room: string
content: text
tag: string
Description: Send an Entry to a room with content, tag

Link: /entry/byid?id=?id=?id...
Method: GET
id: 64 bit string ID, can request for multi id at a time
Description: Get entries with id string

Game Server Account request

Link: /user/?id=?id=?id...
Method: GET
id: 64 bit string ID, can request for multi id at a time
Description: Get user detail with id string

Link: /user/create?name=?password=
Method: POST
name: string
password: string
Description: Create an user with user name and password, point = 0

Link: /user/getid?name=?password=
Method: GET
name: string
password: string
Description: Get user id by user name and password

Link: /user/modify?id=?password=?point=
Method: POST
id: 64 bit string ID
password: string
point: [sign]+int, if sign=='+', add point to current point, same for '-','*','/', if no sign , set new point = point
Description: Modify an user detail by id

Game Server Image request

Link: /image/[img_id.png]?img_id=?name=?thumb=
Method: GET
img_id: 64 bit string ID
name: image name
thumb: thumb=yes, return thumb instead of image
Description: Get an image with img_id string or image name

Link: /image/create/?name=?content=?thumb=
Method: POST
name: string
content: base 64 string represent an image
thumb: if thumb, make thumb for image. if thumb=content, use thumb for content
Description: Upload an image to server with name (use /image/upload/for test );
